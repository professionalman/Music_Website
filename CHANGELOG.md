## [2.0.0] - 2025-07-17 - The Full-Stack Rebirth

This is the largest update in the history of MyMusic Player, marking its transformation from a static web project into a complete and robust full-stack web application.

### Added
- **New Backend & Database:** Built a brand-new backend with **Node.js** and **Express.js**, integrated with **MongoDB** to dynamically manage all data.
- **RESTful API System:** Established a complete API for efficient client-server communication.
- **User Authentication System:** Implemented user **Registration** and **Login** with secure password hashing (**bcrypt**) and session management via **JSON Web Tokens (JWT)**.
- **User Personalization:**
  - User-specific **"Liked Songs"** feature.
  - **Account Settings** page with profile updates, avatar uploads, and password changes.
- **Server-Side Rendering:** Converted all static HTML pages to the **EJS** template engine.

### Changed
- All pages now fetch data directly from the backend API instead of static JavaScript files.
- Refactored the entire application architecture to support a full-stack model.

### Added

- Expanded the **Electronic & EDM** playlist with iconic tracks from artists like Alan Walker, K-391, Cartoon, and more.

## [1.4.1] - 2025-07-03 - Footer Rendering Hotfix

### Fixed
- Addressed a bug where the site-wide footer would fail to render on dynamically generated pages (Homepage, Playlist) due to DOM manipulation conflicts.
- Implemented a more robust, dual-strategy approach for footer rendering to ensure consistency across all pages.

## [1.4.0] - 2025-06-25 - Site-Wide Footer & Policy Pages

### Added
- A new, consistent site-wide footer to all pages, generated by a central UI module.
- New static pages for **Legal**, **Privacy Policy**, and **Cookie Policy** to enhance transparency.
- A global CSS class (`.text-content-page`) to standardize the styling of text-heavy pages for better readability.

## [1.3.0] - 2025-06-18 - The Discovery Hub Update

### Added
- **New Homepage Layout:** Transformed the homepage into a dynamic "Discovery Hub" with multiple widgets (Daily Mix, Top Artists, Featured Playlists).
- **"All Playlists" Page:** A new dedicated page (`all_playlists.html`) to browse all available playlists.

### Changed
- Refactored `main.js` into a modular rendering system for the new homepage.

---

## [1.2.0] - 2025-06-11 - Dynamic Artist Pages

### Added
- **Dynamically Generated Artist Pages:** Artist pages are now generated automatically from the song library, eliminating the need for a separate `data/artists.js` file.
- **Context-Aware Playback:** The playing queue is now context-aware, improving the "Next/Previous" experience on artist pages.

### Removed
- Deleted the static `data/artists.js` file, establishing a single source of truth for data.

---

## [1.1.0] - 2025-06-04 - Native Media Integration & Fullscreen Mode

### Added
- **Native Media Control Integration:** Implemented the **Media Session API** for lock screen and hardware media controls.
- **"Now Playing" Fullscreen Mode:** An immersive fullscreen view for the currently playing track.

### Fixed
- Improved the favicon update mechanism for better compatibility with Safari/iOS.
