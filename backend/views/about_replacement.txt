                <h1 class="text-center mb-4">About MyMusic Player</h1>

                <p class="lead text-center mb-5">
                    MyMusic Player is an online music streaming platform designed and developed to provide a simple,
                    modern, and intuitive listening experience, inspired by popular platforms like Spotify and Apple Music.
                </p>

                <section class="mb-5">
                    <h2 class="h4 border-bottom pb-2 mb-3">Project Goals</h2>
                    <p>The main objective of the project is to build a full-stack web application showcasing the ability to develop
                        both user interface (UI/UX) and backend functionality for a music streaming platform, focusing on:</p>
                    <ul>
                        <li><strong>Interface Design:</strong> Create a clean, aesthetically pleasing, and user-friendly layout.</li>
                        <li><strong>Core Functionality:</strong> Implement essential music player features like play/pause,
                            progress bar, and volume control.</li>
                        <li><strong>Content Management:</strong> Build an admin panel for managing songs, artists, and playlists dynamically.</li>
                        <li><strong>User Authentication:</strong> Implement secure login system with role-based access control (admin, artist, user).</li>
                        <li><strong>Responsive Design:</strong> Ensure the app adapts well across devices, from desktop to mobile.</li>
                        <li><strong>Dynamic Interactions:</strong> Use JavaScript for smooth, interactive UI updates.</li>
                        <li><strong>Database Integration:</strong> Store and retrieve music data, user information, and playlists from MongoDB.</li>
                        <li><strong>File Upload System:</strong> Allow admins to upload audio files and cover images.</li>
                        <li><strong>Advanced Controls:</strong> Add familiar playback options like shuffle and repeat modes.</li>
                    </ul>
                    <p><em>The project serves as hands-on practice to strengthen full-stack development skills.</em></p>
                </section>

                <section class="mb-5">
                    <h2 class="h4 border-bottom pb-2 mb-3">Technology Stack</h2>
                    <p>MyMusic Player is built with modern web technologies:</p>
                    <ul>
                        <li><strong>Frontend:</strong>
                            <ul>
                                <li>HTML5, CSS3 (Flexbox, Grid, CSS variables)</li>
                                <li>JavaScript (ES6+) for dynamic interactions</li>
                                <li>EJS templating engine for server-side rendering</li>
                            </ul>
                        </li>
                        <li><strong>Backend:</strong>
                            <ul>
                                <li>Node.js with Express.js framework</li>
                                <li>MongoDB with Mongoose ODM</li>
                                <li>JWT authentication for secure access</li>
                                <li>Multer for file upload handling</li>
                            </ul>
                        </li>
                        <li><strong>Features:</strong>
                            <ul>
                                <li>RESTful API architecture</li>
                                <li>Role-based access control (RBAC)</li>
                                <li>File storage and management system</li>
                                <li>Dynamic playlist and artist management</li>
                            </ul>
                        </li>
                    </ul>
                </section>

                <section class="mb-5">
                    <h2 class="h4 border-bottom pb-2 mb-3">Key Features</h2>
                    <ul>
                        <li><strong>User Features:</strong>
                            <ul>
                                <li>Browse songs by playlists and artists</li>
                                <li>Search functionality for songs and artists</li>
                                <li>Create and manage personal playlists</li>
                                <li>Mark songs as favorites</li>
                                <li>Full music player controls with queue management</li>
                            </ul>
                        </li>
                        <li><strong>Admin Features:</strong>
                            <ul>
                                <li>Upload and manage songs with audio files and cover art</li>
                                <li>Create and manage artist profiles</li>
                                <li>Edit and delete existing content</li>
                                <li>View and organize all music library</li>
                            </ul>
                        </li>
                        <li><strong>Artist Features:</strong>
                            <ul>
                                <li>View personal artist profile</li>
                                <li>Track songs associated with their profile</li>
                            </ul>
                        </li>
                    </ul>
                </section>
